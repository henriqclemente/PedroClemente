## Sobre mim

Construo produtos SaaS reais utilizando **orquestração de LLMs e prompt engineering** como principal motor de desenvolvimento. Minha abordagem não é a de um dev tradicional — sou um **AI orchestrator**: defino arquitetura, estruturo fluxos e entrego produtos completos com velocidade, usando IA como camada de execução.
